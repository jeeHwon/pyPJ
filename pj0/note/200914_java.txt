# 웹문서 작성 기본
 --front side & back side
  1) HTML : 골격, 내용
  2) CSS : 웹 서식
  3) Javascript : 유효성, 무결성 data 수집 위한 value 값 체크
  4) jQuerry : Javascript 위한 라이브러리(엘리먼트 효율적 제어)
  5) JSP(JavaServerPage) : Java 기반 Serverside 스크립트 언어(HTML 에 Java 언어 들어가 있으면)
  6) Servlet : Java에 HTML 들어가있으면
  7) XML : 확장될수 있는 표시 언어 (임의로 태그 생성 가능)
  8) Ajax : Javascript 위한 라이브러리(일부 데이터 로드 기법, 자동완성 기능 등)
  8) Json : 경량의 Data 교환 형식 (Javascript 객체 만들 때 사용하는 표현식)
